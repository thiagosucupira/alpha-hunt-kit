# Public release checklist

- [ ] `pytest -q` passes.
- [ ] `scripts/validate_public_repo.py` passes.
- [ ] Quickstart works from a clean clone.
- [ ] No `.env`, key, token, broker id, private path, or local runtime state is committed.
- [ ] Optional adapters are documented as optional.
- [ ] README says fixture results do not prove edge.
- [ ] Agent prompt tells agents to promote or bury, not silently optimize.
- [ ] Visual assets render and have alt text in docs/blog.
